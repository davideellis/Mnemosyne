# Mnemosyne

Mnemosyne is an open-source, local-first notes application for iOS, macOS, Windows, and Android.

The product goal is simple: your notes live as plain Markdown files in a folder you choose, stay fully usable offline, and sync securely across your personal devices when you sign in to the same account.

## Why The Name?

Mnemosyne is the Greek goddess of memory and remembrance. The name fits a notes system built around preserving what matters in a form you control.
